# Recipe REST API
####  Recipe REST API is an advanced REST API for managing recipes created with Python, Django REST Framework and Docker using Test Driven Development.

## Project overview
- CRUDs for Users, Recipes, Tags, Ingredients, Images
- User authentication (token authentication)
- Filtering and sorting objects
- Uploading and viewing images
- 10 endpoints with documentation in **OpenAPI 3** format via drf-spectacular
- 57 unit tests
